## Description
A multi-tool library mod for mods made by the Mystic Creations Team.
<br><br>
MysticLib is a library of shared code between most of our projects. Which includes:
 - Config Builder for easy config creation, with a built-in editor in-game
 - WorldMixer
 - Variable Utils
 - Datagen Utils
 - Command Utils
