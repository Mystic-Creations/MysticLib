## Description
MysticLib is a library mod for all most used code in projects by Mystic Creations Team.
<br><br>
This library contains such as e.g. many useful util classes (Entity, Level, Ticking, Datagen etc),
config library and variable management (Both cross-loader and Fabric Data Attachments API since 1.20.4).<br>
It is cross-loader, and multi-version but only for loaders and versions we develop on.

## Developing with MysticLib
MysticLib has a wide documentation on how to set it up and how to work with it.<br>
